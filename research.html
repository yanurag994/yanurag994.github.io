Helix Algorithm for Delaunay Triangulation

  • Developed Helix Algorithm for Construction of Delaunay Triangulation in High Dimensions on reduced memory footprint,
  • C++ Implementation for Single Core, Multi Core, and Distributed Architectures.
  • Utilization of branchless programming principles, I/O optimization for speedup,
  • Achieved over 10 to 15 times reduction in memory consumption allowing processing of significantly larger datasets,
  • Complete elimination of any hard limits (out of memory) by modifying algorithm for partitioning approach.

Incremental Critical Cells Algorithm

  • Implementation of incremental Critical Cells algorithm in C++, along with language specific optimization.
  • Updated Code Base for template-based constructions allowing user portability in terms of input types of complex and distance matrices.
  • Study on the effect of parallelization across different scenarios and identification of best parallelization strategy.
